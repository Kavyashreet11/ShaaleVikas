# Shaale-Vikas – School Alumni Support System
 
## Project Overview
Shaale-Vikas is an Android application developed to connect rural schools with alumni and local supporters through a transparent digital support platform. Many rural schools face infrastructure problems such as damaged classrooms, broken benches, poor sanitation facilities, and delayed repair funding. Alumni are willing to support schools but often do not know the current needs of the institution.
 
Shaale-Vikas acts as a bridge between schools and alumni by allowing school administrators to add repair and infrastructure needs while supporters can view requirements and pledge support digitally.
 
---
 
## Problem Statement
Rural schools frequently face infrastructure maintenance issues due to delayed government funding. Problems such as classroom damage, broken benches, toilet repair issues, and poor learning environments directly affect students. There is no transparent platform that allows alumni and supporters to view and contribute towards current school needs.
 
The Shaale-Vikas application solves this issue by creating a digital school-alumni support system with real-time updates and transparent progress tracking.
 
---
 
## Objectives
- Create a digital platform for rural school infrastructure support
- Connect alumni and supporters with school development activities
- Provide transparency using progress tracking and donor information
- Simulate support pledges without real payment integration

## Modules
 
### 1. Needs Dashboard
Displays all school repair requirements with images, estimated cost, progress bar, and pledge support button.
 
### 2. Add Need Module
Allows admin/headmaster to add new infrastructure needs dynamically.
 
### 3. Progress Tracking
Progress bars update dynamically whenever support is pledged.
 
### 4. Donor Hall of Fame
Displays names of alumni and contributors who supported the school.
 
### 5. Impact Photos
Shows before and after repair images for transparency.
 
### 6. AI Suggestion Module
Uses Google Gemini AI Studio to generate infrastructure improvement suggestions.
 
---
 
## Firebase Integration
Firebase Realtime Database is used to store:
- Need title
- Estimated cost
- Progress percentage
- Need images
 
The application updates data dynamically in real time.
